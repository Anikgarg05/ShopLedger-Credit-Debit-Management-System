# Shop Credit Management System

A full-stack web application to manage customer credit and debit transactions.

## Features
- Add/Delete Customers
- Credit & Debit Tracking
- Audit Dashboard (Chart.js)
- Search functionality

## Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite

## Run Locally

1. Clone repo:
   git clone <your-repo>

2. Install dependencies:
   pip install -r requirements.txt

3. Create .env file:
   OWNER_EMAIL=admin@gmail.com
   OWNER_PASSWORD_HASH=<your hash>

4. Run:
   python app.py
